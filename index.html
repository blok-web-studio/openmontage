How It Works

Your agent is the director.

No code orchestrator. Your AI coding assistant reads pipeline manifests, calls Python tools, self-reviews, and renders the final video. You stay in control at every creative decision.

01

Research

Before writing a single word, the agent searches YouTube, Reddit, Hacker News, news sites, and academic sources. It gathers data points, audience questions, trending angles, and visual references.

02

Proposal

You get 2-3 differentiated concepts with honest tool paths, cost estimates, and a sample before full production starts. Pick one. The agent locks it in.

03

Script & Scene Plan

The agent writes the script with voice direction, plans every scene, and maps assets to providers. Every creative decision gets your approval via the Backlot storyboard.

04

Asset Generation

Images, video clips, narration, music — all generated or sourced automatically. The scored provider selection ranks every tool across 7 dimensions and picks the best match.

05

Compose & Render

Remotion or HyperFrames assembles everything into a finished video. Post-render self-review catches issues before you see the output.

Quick Start

Up and running in 3 commands.

Python 3.10+, FFmpeg, Node.js 18+, and an AI coding assistant. That's it.

terminal
# Clone and install
git clone https://github.com/calesthio/OpenMontage.git
cd OpenMontage
make setup
 
# Open in your AI coding assistant and say:
"Make a 60-second animated explainer about how neural networks learn"
 
# Or with zero API keys — free real-footage documentary:
"Make a 90-second documentary about city life at 4am. Real footage only."

Works with Claude Code, Cursor, Copilot, Windsurf, and Codex — any AI coding assistant that can read files and run code.

Start producing.

Open-source. Free to start. No vendor lock-in. Your agent handles the production — you handle the vision.
